X64EZ is a fork of kubkon's zig-dis-x86_64, that I've switched up a bit to make it more usable for my purposes as an assembler/disassembler for the Ribbon Programming Language . I have added a friendly one-shot disassembler function, and a few quality of life enhancements when using it as an assembler, such as constants utilizing Zig's enum literal coercion.
